# 📚 Student Enrollment Form using JsonPowerDB

---

## 🏷️ Title of the Project

**Student Enrollment Form (CRUD Application using JsonPowerDB & jQuery)**

---

## 📖 Description

This project is a simple web-based application that allows users to manage student enrollment details. It uses **JsonPowerDB (JPDB)** as the backend database and **jQuery + HTML + Bootstrap** for the frontend.

Users can:

* Add new student records
* Retrieve existing student data using Roll-No
* Update student details
* Reset the form

The application interacts with JsonPowerDB using REST APIs, making it lightweight and efficient.

---

## 🚀 Benefits of using JsonPowerDB

* ⚡ **High Performance** – Faster than traditional databases
* 🔌 **REST API Based** – Easy integration with frontend
* 🧩 **Schema-free** – No rigid table structure required
* 📦 **Lightweight** – No complex setup or installation
* 🔐 **Built-in Security** – Uses connection tokens
* 🔄 **Real-time CRUD operations**
* 🛠️ **Developer Friendly** – Works easily with JavaScript/jQuery
